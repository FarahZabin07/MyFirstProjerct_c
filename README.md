# Cricket Management System

A C-based project to manage cricket teams, calculate ratings, and determine Man of the Match.

## Features
- User Login
- Team data from files
- Player rating calculation
- Man of the Match selection

## Files
- src/main.c → Source code
- data/LogIn.txt → Login credentials
- data/Team1.txt → Team 1 data
- data/Team2.txt → Team 2 data
